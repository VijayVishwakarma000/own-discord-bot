A simple Discord bot built with Node.js and discord.js that automatically welcomes new members and sends a farewell message when users leave the server.

Features:

    Sends a welcome message when a user joins the server
    Sends a goodbye message when a user leaves
    Fully customizable messages
    Lightweight and easy to extend
    Built using JavaScript (Node.js)

Tech Stack:

    Node.js
    discord.js
    JavaScript

Prerequisites:

    Before running this bot, make sure you have:
    Node.js v16.9.0 or later
    A Discord account
    A Discord bot token
    Server permissions to add bots
